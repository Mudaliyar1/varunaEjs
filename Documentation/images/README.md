# Diagram Images

This folder will contain PNG versions of the architecture and database schema diagrams after running the conversion script.

## How to Generate PNG Images

1. Install puppeteer:
   ```
   npm install puppeteer
   ```

2. Run the conversion script:
   ```
   node ../convert_to_png.js
   ```

3. The following PNG files will be generated:
   - Project_Architecture_Diagram.png
   - MongoDB_Schema_Diagram.png

These PNG images can be easily included in reports, presentations, or documentation.
