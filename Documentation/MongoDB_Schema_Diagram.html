User Schema

  • username: String (required) - User's display name
  • email: String (required, unique) - User's email address for login
  • password: String (required, hashed) - Securely stored password
  • role: String (enum: ['admin', 'crm', 'ce', 'user'], default: 'user') - Access control role

Suggestion Schema

  • name: String - Customer's full name
  • mobileNo: String - Customer's contact number
  • email: String - Customer's email address
  • state, city, district: String - Customer's location details
  • businessType: String - Type of customer's business
  • findForm: String - How customer found the website
  • typeOfProducts, head, flow, pipeSize, phase, frequency: String - Product requirements
  • createdAt: Date (default: Date.now) - Submission timestamp
  • sentToCE: Boolean (default: false) - Whether assigned to CE
  • ceAssigned: ObjectId (ref: 'User') - Reference to CE user
  • completed: Boolean (default: false) - Whether inquiry is resolved
  • message: String - Additional notes

Product Schema

  • Basic Information:
    • model: String - Product model identifier
    • type: String - Type of product
    • description: String - Product description
    • input: String - Main product image URL
    • images: [String] - Additional product images
  • Product Identification:
    • pump_id: String - Unique identifier
    • pump_app_name: String - Application name
    • pump_model_name: String - Model name
  • Performance Specifications:
    • stages: Number - Number of pump stages
    • head_meters: [Number] - Head range in meters
    • discharge_lpm: [Number] - Flow rate in LPM
    • pump_discharge_type, pump_rated_head: String - Discharge specifications
  • Physical & Electrical Details:
    • motor_rating: String - Motor rating
    • pump_voltage, pump_frequency, pump_power_supply: String - Electrical specifications
    • suction_size, pump_del_size: String - Physical dimensions
    • material: String - Construction material
  • reviews: [ReviewSchema] - Embedded customer reviews
  • created_at, updated_at: Date - Timestamps
  • created_by: String - Admin who created the product

Review Schema (Embedded in Product)

  • user: String (required) - Reviewer's name
  • rating: Number (required, min: 1, max: 5) - Star rating
  • comment: String (required) - Review text
  • createdAt: Date (default: Date.now) - Review timestamp
